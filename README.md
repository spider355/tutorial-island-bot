# Tutorial Island Bot

Automated Tutorial Island completion plugin for RuneLite/Microbot.

## Features
- ✅ Character creation (female)
- ✅ Random name generation
- ✅ All 12 tutorial stages automated
- ✅ Ironman support (Regular/Hardcore/Ultimate)
- ✅ Walk to Falador bank after completion
- ✅ Error recovery (10 retries)
- ✅ Resume from any stage

## Installation

### Pre-built JAR
1. Download `tutorial-island-bot-1.0.0.jar` from [Releases](https://github.com/YOUR-USERNAME/tutorial-island-bot/releases)
2. Place in `~/.runelite/plugins/` (Windows: `%USERPROFILE%\.runelite\plugins\`)
3. Restart RuneLite
4. Enable "Tutorial Island Bot" in settings

### Build from Source
**Windows:**
